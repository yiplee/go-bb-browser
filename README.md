# go-bb-browser

Private experiment: a **Go** reimplementation inspired by [bb-browser](https://github.com/epiral/bb-browser) (CLI + local daemon controlling a real browser session), with the browser layer driven by **[chromedp](https://github.com/chromedp/chromedp)** (Chrome DevTools Protocol).

Implementation strategy and milestones are documented in:

- [`docs/IMPLEMENTATION_PLAN.md`](docs/IMPLEMENTATION_PLAN.md)

No application code yet — planning phase only.
